HTML(Hyper text Mark up Language)

BASIC TAGS

The heading tag
< h1 > Write the heading < /h1 >
For semiheading < h2 > tag furthur h3,h4,h5,h6 are there

Paragraph tag
< p > Write the content < /p >

Can take number of these tags according to the requirement

STYLE OF TEXT

These are opening and closing tags

For making the text bold
< b> Content < /b >
OR
< strong > Content < /strong >

For making text italic
< i> Content < /i >
OR
< em> Content < /em >

For underlined text >
< u> Content < /u >

OR
< ins> Content < /ins >

For Superscript
< sup> Content < /sup >
For Subscript
< sub > Content < /sub >
For Strike
< strike > Content < /strike >
OR
< del> Content < /del >

Most common single opening and closing tag

For horizontal line < hr >
For breaking line or jumping to next line < br >

LIST

There are two types of list
1)Unordered list
2)Ordered list.,

Unordered list is followed by opening and closing tags as
< ul >
< li > Content1 < /li >
< li > Content2 < /li >
< /ul >|

Ordered list is followed by opening and closing tags as
< ol >
< li > Content1 < /li >
< li > Content2 < /li >
< /ol >|

BASIC

< body bgcolor="Color_name" >

< img src="name.type" width="_px";align="_" >
