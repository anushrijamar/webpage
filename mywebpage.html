CLASS AND ID's

Class and ID's ;class can be written multiple times while id can be used once

class is defined inside the particular h or p as:
< p class="any name" >|
while in head section it is defined as
.class name {
//your requirement like
color:_;
}

ID's is defined inside the particular h or p as:
< p id="any name" >
while in head section it is defined as
#class name {
//your requirement like
color:_;
}
